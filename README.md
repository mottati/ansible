# ansible

Simple sequence loop:

Example:

```
ansible-playbook -i localhost, class-list.yml  -e class_size=2

PLAY [all] *****************************************************************************************************************************************************************************************************************

TASK [Dump Class List] *****************************************************************************************************************************************************************************************************
ok: [localhost] => (item=1) => {
    "msg": "1"
}
ok: [localhost] => (item=2) => {
    "msg": "2"
}

PLAY RECAP *****************************************************************************************************************************************************************************************************************
localhost                  : ok=1    changed=0    unreachable=0    failed=0    skipped=0    rescued=0    ignored=0
```
